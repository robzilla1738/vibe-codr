# Vibe Codr Privacy Policy

Effective: July 14, 2026

Vibe Codr is a local desktop client for the open-source vibe-codr coding agent.
The Vibe Codr project does not operate an advertising, analytics, telemetry, or
remote crash-reporting service, and it does not sell personal information.

## Data the app uses

Vibe Codr can access the projects, files, terminal sessions, Git repositories,
and local configuration that you choose to use with the app. Session history,
settings, credentials, and local crash breadcrumbs remain on your device unless
you choose an external service or share that information yourself.

## External services

When you configure and use an AI provider, MCP server, Git remote, or another
external integration, Vibe Codr sends the information needed to complete your
request directly to that service. This may include prompts, selected file
content, tool input, repository data, and service credentials. Each external
service processes that information under its own terms and privacy policy. You
control which services are configured and when they are used.

## Update checks

Installed macOS and Windows builds contact GitHub Releases after launch and
when you choose Help → Check for Updates. The request includes the information
normally sent with an HTTPS request and the app's current version so GitHub can
return the correct release metadata. Vibe Codr does not use update checks for
analytics or advertising, and updates require your consent before download and
restart.

## Storage and retention

App data is stored locally on your device and remains there until you remove it,
clear the applicable session or setting, uninstall the app, or delete its local
application data. Removing the app does not delete your project files.

## Children

Vibe Codr is a developer tool and is not directed to children.

## Changes and contact

Material changes to this policy will be published in this repository with a new
effective date. Questions or privacy requests can be opened at
https://github.com/robzilla1738/vibe-codr/issues.
